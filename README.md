# Inventory Management Dashboard (Excel)

## Project Overview

This project presents an interactive **Inventory Management Dashboard** built using Microsoft Excel.  
The dashboard helps analyze product sales, inventory performance, and regional sales distribution.

The dashboard enables business users to monitor key performance indicators (KPIs) and identify top-performing products and regions.

---

## Objectives

- Track sales performance
- Analyze regional sales distribution
- Identify top-selling products
- Monitor inventory quantity
- Provide interactive filtering using slicers

---

## Dataset Information

The dataset contains **400 records** with the following information:

- Product category
- Product name
- Order quantity
- Sales price
- Profit
- Region
- Customer status
- Order date

Key Columns:

- RegionName
- CategoryName
- ProductName
- OrderItemQuantity
- PerUnitPrice
- Profit
- Month

---

## Dashboard Features

### KPI Metrics

- Total Sales
- Total Profit
- Total Quantity Sold
- Average Profit

### Visualizations

1. Monthly Sales Trend (Line Chart)
2. Sales by Category (Bar Chart)
3. Sales Distribution by Region (Pie Chart)
4. Top 10 Products by Sales (Column Chart)

### Interactive Filters

- Month
- Category
- Region
- Customer Status

## Key Insights

- Some regions generate significantly higher revenue.
- A small number of products contribute to most of the sales.
- Monthly trends help identify peak demand periods.
- Category-wise analysis shows which product groups perform best.
